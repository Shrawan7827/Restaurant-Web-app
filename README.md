# 🍽️ Restaurant Web App

A responsive and modern restaurant website built using **HTML, CSS, and JavaScript**.  
Users can explore food items, view categories, and place orders easily.

---

## 🚀 Features

- 🏠 Home page with attractive UI
- 🍕 Food categories (Pizza, Burger, Momo, etc.)
- 📋 Food menu with images and pricing
- 🛒 Order page with form
- 🔄 Dynamic food selection (image + price change)
- ✅ Order success message on submission
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How to Run the Project

1.  Run `index.html` in your browser
